# Fullstack Ecommerce System with AI Integration
A modern, full-stack e-commerce platform built with the latest technologies.
## 🚀 Technologies
- **Frontend**: Next.js (React 19), Tailwind CSS, Lucide Icons.
- **Backend**: Laravel (PHP), RESTful API.
- **AI**: Gemini AI API (Customer Support Chatbot).
- **Database**: MySQL / SQLite.
## ✨ Key Features
- **AI Chatbot**: Intelligent automated support for customers.
- **Dynamic Dashboard**: Real-time sales and inventory statistics.
- **Product Management**: Multi-category, brands, and attribute system.
- **Responsive UI**: Optimized for all devices using modern CSS.
## 🛠️ Installation
1. Clone the repository.
2. Install dependencies: `npm install` (frontend) & `composer install` (backend).
3. Set up environment variables in `.env`.
4. Run migrations: `php artisan migrate`.
